# Impact-of-Climate-Change-on-Global-Food-Production

**Overview**:\
This project analyzes how climate change variables—specifically temperature and drought patterns —affect global food production. Using statistical modeling and data visualization in R, we explore relationships between environmental changes and agricultural output across different countries and regions.

**Objectives**:
- Examine trends in temperature and precipitation over time\
- Analyze their impact on crop yields\
- Identify regions most vulnerable to climate change\
- Build predictive models for food production

**Tools & Technologies**:
- R
- RStudio
- GitHub (version control & collaboration)
